# Gaming PC Build Documentation

## Overview

This repository documents my custom gaming PC build, including the hardware used, the assembly progress, and the final completed setup.

The system is built around an **AMD Ryzen 7 9700X** and a **Zotac GeForce RTX 5070 Ti Solid SFF OC 16GB**, using a white-themed aesthetic inside an **NZXT H6 Flow**. Cooling is handled by a **Thermalright Peerless Assassin 120 SE ARGB**, and the build also includes a **Lian Li LCD screen** mounted inside the case for system monitoring and visual customization.

The photos currently included in this documentation show:
- the motherboard installed in the case
- the CPU cooler installation process
- GPU installation
- rear cable management and SSD mounting
- the completed build powered on with blue lighting
- the LCD display installed and functioning

## Build Progress Summary

Based on the current photos, the build process included:

1. Installing the motherboard into the NZXT H6 Flow.
2. Preparing for CPU cooling and thermal paste application.
3. Installing the Thermalright Peerless Assassin 120 SE ARGB cooler.
4. Installing the Zotac RTX 5070 Ti graphics card.
5. Connecting the white sleeved power cables.
6. Routing cables in the rear chamber.
7. Mounting the secondary 500GB SSD in the back of the case.
8. Installing and powering the LCD screen accessory.
9. Completing the system and confirming successful boot.

The final photos show the PC running with blue lighting, the LCD panel active, and the system fully assembled.

## Setup Notes

After physical assembly, the system setup includes:

1. Enter BIOS and confirm all hardware is detected.
2. Enable **EXPO** so the RAM runs at DDR5-6000.
3. Install the operating system on the **Samsung 990 Pro 2TB NVMe SSD**.
4. Install motherboard chipset and device drivers.
5. Install NVIDIA GPU drivers.
6. Install software for the LCD screen accessory.
7. Confirm both storage drives are visible in BIOS and in the operating system.
8. Check fan behavior, temperatures, and lighting configuration.
